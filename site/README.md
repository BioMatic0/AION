# ∞ Holding / Wirth Nexara Website

Statische, responsive Mehrseiten-Website.

## Start

Öffne `index.html` direkt oder starte lokal:

```bash
python3 -m http.server 8000
```

Dann: http://localhost:8000

## Struktur
- 7 Gründerprofile
- 7 Lebenswerke
- 1 Holding-Vision
- 1 Vertrauensperson als Fundament ohne eigenes Werk

## Technik
Reines HTML, CSS und JavaScript. Keine externen Abhängigkeiten.
