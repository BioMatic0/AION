# Security Policy

## Reporting a vulnerability

If you find a security issue, do not publish exploit details immediately in a public issue.

Use the project reference contact first:

- Patrick Wirth
- patrickwirth_93@icloud.com

Include:

- affected component
- reproduction steps
- impact
- proposed mitigation if available

## Scope note

This repository currently contains MVP and pre-release components. Some areas are still
under active development and may not yet meet hardened production standards.
