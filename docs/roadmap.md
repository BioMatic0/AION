# AION Roadmap

## Current public baseline

- Public GitHub repository
- Public releases for Windows and Android plus iOS Xcode project
- Open MIT license
- Contribution, code-of-conduct and security documents
- Buildable monorepo with API, web and desktop surfaces

## Near-term focus

### 1. Product hardening

- stronger auth and per-user data separation
- deeper test coverage
- packaging and runtime polish for desktop and mobile

### 2. AI depth

- external model provider integration
- embeddings and semantic retrieval
- stronger search and memory linking

### 3. Operations

- backup and restore workflows
- observability and error tracking
- cleaner deployment automation

## Later expansion

- voice
- media
- browser and deep search
- collaboration
- messenger
- cloud and extensions
