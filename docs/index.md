# AION

Open, human-centered AI platform for reflection, structure, privacy and growth.

## Project goals

- Keep the software freely accessible
- Make contribution easy and public
- Build a trustworthy foundation for reflection, structure and personal growth
- Keep governance, privacy and safety visible in the product

## Open collaboration

AION is published as an open MIT-licensed project with public source access and
shared contribution files already included in the repository:

- `LICENSE`
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `SECURITY.md`

- Patrick Wirth
- patrickwirth_93@icloud.com

## Current scope

The repository currently includes:

- Next.js web app
- NestJS API
- shared packages for governance, prompts, AI core and UI
- Windows packaging
- Android build output
- iOS Xcode project export
- visible governance, privacy and security surfaces

## Build status

The source repository includes CI and test configuration. Public releases are
published through GitHub Releases.
