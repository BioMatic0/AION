# AION v0.1.0

First public release of AION as an open, human-centered platform for reflection,
structure, privacy and growth.

## Included artifacts

- Windows installer
- Windows portable build
- Android APK
- iOS Xcode project
- release bundle
- SHA256 checksums

## Public links

- Source repository: <https://github.com/BioMatic0/AION>
- Latest release: <https://github.com/BioMatic0/AION/releases/latest>
- Project page: <https://biomatic0.github.io/AION/>
- Discussions: <https://github.com/BioMatic0/AION/discussions>

## Notes

- Windows can be used directly through the installer or the portable build.
- Android is currently distributed as a debug build.
- iOS requires macOS and Xcode to create a signed mobile build from the provided project.
- The source code is publicly available and released under the MIT License.
