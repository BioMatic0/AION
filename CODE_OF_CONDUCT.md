# Code of Conduct

## Our commitment

We want AION to be a respectful, constructive and open collaboration space.

Participants are expected to:

- treat others with respect
- discuss ideas directly without personal attacks
- accept critique without escalating into hostility
- avoid harassment, intimidation, humiliation or exclusion
- avoid abusive, hateful or discriminatory behavior

## Project standard

This project is meant to support shared contribution for a better future and
human connection. Behavior that undermines that purpose can be moderated or removed.

## Enforcement

Project maintainers may remove comments, issues, pull requests or contributions
that seriously violate these standards.
